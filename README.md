# Employee Management System (Java)

A console-based Employee Management System built using Java and OOP concepts.

## 🔧 Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Uses ArrayList for storage

## 🛠️ Technologies Used
- Java
- OOP (Encapsulation, Classes, Objects)
- ArrayList

## ▶️ How to Run
1. Compile:
   javac Main.java

2. Run:
   java Main
